## Woulda Coulda Shoulda

To use run bin/wcs with a url for an ESPN Team

```ruby
$ bin/wcs "http://games.espn.go.com/ffl/clubhouse?leagueId=186439&teamId=2&seasonId=2014"
=> +------------+------------+
  | Hemingway's Horde (GAR) |
  +------------+------------+
  | Woulda     | 57.8       |
  | Coulda     | 212.8      |
  | Shoulda    | 144.9      |
  +------------+------------+
```
